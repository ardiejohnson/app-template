# app-template

Starter for new apps in the ardiejohnson.com portfolio.

**Start a new app from any device (phone included):**
1. Click **"Use this template" -> Create a new repository** above, and name it (e.g. `moodboard`).
2. Open Claude Code on the new repo and hand the **new-app** agent your HTML or JSX file.
   It scaffolds the project, builds it, and gets it ready to preview.
3. Import the new repo into Vercel once and attach its subdomain.

This repo ships with the standard agents (`.claude/agents/`) and `CLAUDE.md`, so every
new app is born already wired into the system.
